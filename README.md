## О проекте

Данный проект реализует алгоритм электронной цифровой подписи по стандарту  
**ГОСТ Р 34.10-2012** на Python с использованием собственной криптографической реализации.

Он позволяет генерировать ключи, создавать и проверять цифровые подписи файлов и сообщений.

---

## Возможности

- Генерация ключевой пары (private / public key)
- Создание цифровой подписи
- Проверка подписи
- Работа с файлами и сообщениями
- Использование собственной реализации криптографии (PyGOST)
- Кроссплатформенность (Linux / Windows)
